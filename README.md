## 为何公开（Public）

本仓库选择 **Public** 而非 Private，原因如下：

1. **内容均为公开信息整理** — 来源为 Bloomberg、郭明錤、MacRumors、产业链公开报道等，不含苹果内部机密或未公开 NDA 资料。
2. **学习与研究用途** — 面向硬件爱好者、供应链从业者与 AI 可穿戴方向的学习者，公开便于检索、引用与协作补充。
3. **已做免责声明** — 全文标注「传闻 / 推演 / 置信度」，不构成投资或采购建议，公开传播风险可控。
4. **模块化结构便于社区贡献** — 按 BOM 子系统拆分，后续可接受 Issue / PR 纠错与数据更新。
