# 三层算力架构 · 成本区间汇总

## 1. 结论（单副眼镜 · 该模块视角）

| 指标 | 数值 |
|------|------|
| **眼镜端 BOM 区间** | $105 – $155 |
| **中位置信** | ~$126 |
| **占 Meta 对标比** | ~93%（Meta ~$135） |

三层算力使 **镜端硅片成本可低于 Meta AR1 路线**（算力外置 iPhone），但 **结构/光学/EMS 仍接近传统眼镜上限**。

## 2. 成本结构（眼镜端）

- 算力+存储+连接：~49%
- 结构+EMS：~28%
- 视觉模组：~14%
- 声学+电源：~9%
