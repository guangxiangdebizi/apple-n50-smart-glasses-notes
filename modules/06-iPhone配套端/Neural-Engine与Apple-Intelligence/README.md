# Neural Engine 与 Apple Intelligence

> **免责声明**：NE 规格来自苹果已发布芯片；N50 任务分配为产业链推演，**非官方**。

## 模块概述

N50 **重 AI 推理卸载至 iPhone Neural Engine**。眼镜 N401 仅唤醒/预筛选；Visual Intelligence、Foundation Models、Siri 2.0 在 iPhone 端执行。

| 科研文档 | 内容 |
|----------|------|
| [01-最小BOM单元拆解](./科研/01-最小BOM单元拆解.md) | NE die / 内存带宽 / 模型存储单元 |
| [02-单价与用量估算](./科研/02-单价与用量估算.md) | 系统级成本分摊模型 |
| [03-成本区间汇总](./科研/03-成本区间汇总.md) | 摊到单副眼镜的隐性成本 |

**注意**：iPhone 芯片 **不计入眼镜 BOM**；科研文件夹记录「配对生命周期分摊」。
