## 传闻规格

| 协议 | 用途 | 置信度 |
|------|------|--------|
| BT LE | 音频、控制 | 高 |
| UWB | 视觉特征低延迟传输 | 中 |
| 带宽 | 特征流 ~1–5 Mbps | 低 |

**对比 Meta**：Gen 2 用 Wi-Fi + BT，镜端 AR1 更独立。
