## 单模块文档模板

每个 `.md` 文件统一包含以下章节：

1. **模块概述** — 在 N50 中的角色
2. **N50 传闻规格** — 代号、参数、置信度
3. **核心供应商与竞争格局** — 表格：厂商 / 份额或地位 / 与苹果关系
4. **供应链逻辑** — 上游→中游→苹果→下游，关键节点与议价权
5. **厂商优势分析** — 为何选这家、技术/规模/果链绑定
6. **前沿未解问题** — 行业级技术瓶颈（非仅 N50）
7. **关键参考数据** — 市场规模、功耗、成本、出货量等（附来源链接）
8. **延伸阅读** — 论文、研报、新闻
