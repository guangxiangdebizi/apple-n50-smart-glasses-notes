# ISP · 图像信号处理

---

## 1. 模块概述

N50 传闻配备 **主摄（5–8MP、9:16 竖向）+ 副摄（低分辨率广角/CV）**。苹果 **自研 ISP** 嵌入 N401 或紧耦合协处理器，负责 **HDR、降噪、隐私指示灯联动、端侧预筛选**（人脸/场景/运动检测），仅将高价值帧或特征送至 iPhone 做 Visual Intelligence 深度推理。

---

## 2. N50 传闻规格（含置信度）

| 项目 | 传闻规格 | 置信度 |
|------|----------|--------|
| ISP 归属 | 苹果自研，集成于 N401 体系 | 中 |
| 主摄输出 | 5–8MP，竖向 9:16，HDR，4K 录制说法 | 中 |
| 副摄职能 | 手势、环境上下文、低功耗常开 | 中 |
| 端侧 AI | 预筛选 + 唤醒，非完整大模型 | 高 |
| 隐私 | 拍摄指示灯与 ISP 管线联动 | 中高 |
| CMOS 供应商 | 索尼定制微型传感器（另见视觉模组文档） | 中 |

---

## 3. 核心供应商与竞争格局

| 环节 | 公司 | 地位 |
|------|------|------|
| ISP 设计 | **Apple** | iPhone/A 系列同源，垂直封闭 |
| 竞品 ISP | **Qualcomm Spectra**（AR1） | Meta 眼镜主流 |
| 竞品 ISP | **SigmaStar SSC309** 等 | 国产眼镜，~1.5 TOPS |
| CMOS | **Sony** | 移动端 CIS 约 40%+ 高端份额 |
| 镜头/模组 | **大立光、玉晶光、高伟** | 果链光学 |

*索尼 CIS：行业普遍估计苹果高端机型主供；ISP 无第三方披露。*

---

## 4. 供应链逻辑

```
Sony CMOS wafer → 模组厂（高伟等）镜头贴合 → FPC/排线
    → 苹果 N401 ISP 驱动固件 → 特征流蓝牙/UWB → iPhone Neural Engine
```

ISP 逻辑在 **苹果硅内闭环**；模组厂不接触 ISP 微码。算法更新经 **iOS 27 + 眼镜固件 OTA** 协同。
