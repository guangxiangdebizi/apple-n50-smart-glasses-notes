# 内存-LPDDR共封装 · 最小 BOM 单元拆解

## 1. 拆解树

```
LPDDR PoP 子系统
├── LPDDR5X DRAM die（4GB 或 8GB）
├── PoP 叠封增量（基板/回流/测试）
├── 与 N401 logic die 互联（短距 BGA）
└── 备选 ePOP（+eMMC，Watch 路线）
```
