# 内存 · LPDDR 共封装（PoP）

> 基于公开传闻整理，**非苹果官方**。

---

## 1. 模块概述

眼镜端内存需在 **亚毫米厚度、数百 mW 功耗** 内为 ISP 缓冲与唤醒提供带宽。传闻 N401 采用 **定制 LPDDR5X**，以 **PoP** 叠封于 SoC 之上——与 Apple Watch SiP 同族。

---

## 2. N50 传闻规格

| 项目 | 传闻/推演 | 置信度 |
|------|-----------|--------|
| 类型 | LPDDR5X | 中 |
| 封装 | PoP 与 N401 共封装 | 中 |
| 容量 | 4–8 GB 级 | 低–中 |
| 供应商 | 三星、SK 海力士 | 中 |

---

## 3. 核心供应商

---

## 4. 供应链逻辑

```
DRAM die（三星/SK）→ 测试 → PoP 叠封（与 SoC）→ SiP → EMS
```

苹果惯用双供应商；眼镜量级小于 iPhone，但定制时序可能单一厂先期独占。

---

## 5. 厂商优势

- **Samsung/SK**：LPDDR5X 旗舰方案；苹果长期合约
- **Apple**：PoP 布局控制 SI；固件级休眠调度
