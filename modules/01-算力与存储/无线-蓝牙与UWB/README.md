# 无线 · 蓝牙与 UWB

> 基于公开传闻整理，**非苹果官方**。

---

## 1. 模块概述

N50 与 iPhone 构成 **分层算力系统**，无线链路承担 **音频、通知、压缩视频/特征流、空间锚定**。**蓝牙**为确定能力；**UWB** 为产业链推演，可能延续 **U1 芯片** 生态。

---

## 2. N50 传闻规格

| 项目 | 传闻/推演 | 置信度 |
|------|-----------|--------|
| 蓝牙 | 标配，深度 iPhone 配对 | 高 |
| UWB | 传闻集成，类似 U1 | 中 |
| Wi-Fi 独立 | 不主打 | 中高 |
| 竞品 | Meta 以 BT + 手机为主 | 高 |

---

## 3. 核心供应商

| 技术 | 主导厂商 | 备注 |
|------|----------|------|
| 蓝牙 | Apple + Broadcom 历史 | 或集成 SiP |
| UWB | Apple U1/U2；Qorvo/NXP | ST 推 802.15.4ab |
| UWB 模组 | Murata、Johanson | 封装天线 |

全球 UWB 市场 2025 约 **$33.7B**（含汽车/IoT 广义口径）。

---

## 4. 供应链逻辑

```
射频 IP → 晶圆封测 → SiP 与 N401 合封 → iPhone UWB 对端
```

UWB 需 iPhone 15+ U1 设备形成网络效应。

---

## 5. 厂商优势

- **Apple**：AirDrop 指向、Car Key、Find My 已验证
- **Qualcomm（对比）**：AR1 单芯片连接成熟，但无苹果级 UWB 生态
