# ISP图像信号处理 · 成本区间汇总

## 1. 结论

| 指标 | 数值 |
|------|------|
| **模块 BOM 区间** | $3 – $18 |
| **中位置信** | ~$8 |
| **占整机 BOM 比** | ~6% |

*已含于 N401 logic die，此处为 **增量归因**，与 SoC 模块不重复加总。*

## 2. 成本结构

- 逻辑 IP 面积（N3E 摊销）：~65%
- NRE/验证摊销：~35%

## 3. 与竞品对标

| 产品 | ISP 成本估 | 来源 |
|------|-----------|------|
| AR1 Spectra | ~$15–20（SoC 内） | $55 反推 |
| N401 自研 ISP | ~$8 增量 | 本模块 |
| iPhone A 系列 ISP | 不可拆分 | 苹果封闭 |

苹果优势：**竖向 9:16 镜筒畸变、隐私指示灯硬联动** 无法外购 IP 复刻。

## 4. 误差与未解问题

1. 自研摊销难量化，NRE 区间 ±100%
2. 双 ISP vs 单 ISP 架构未确认
3. CMOS 成本在 [02-视觉系统](../../02-视觉系统/)，不在本模块
