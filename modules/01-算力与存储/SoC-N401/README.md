# SoC · N401 主控芯片

> 基于公开传闻整理，**非苹果官方**。

---

## 1. 模块概述

**N401** 为苹果智能眼镜内部主控代号，传闻基于 **Apple Watch S 系列 SiP** 裁剪魔改：保留双核 CPU + 4 核 Neural Engine 能效基因，强化 **双摄像头 ISP、蓝牙/UWB 与始终在线唤醒**。

---

## 2. N50 传闻规格

| 项目 | 传闻规格 | 置信度 |
|------|----------|--------|
| 代号 | N401 | 高 |
| 架构基底 | S10/S9 类 SiP | 高 |
| CPU | 64 位双核 | 中 |
| Neural Engine | 4 核 | 中 |
| SoC 功耗上限 | <2 W | 中高 |
| 量产窗口 | 2026 末–2027 | 中高 |

---

## 3. 核心供应商

| 角色 | 公司 | 与 N401 关系 |
|------|------|--------------|
| 设计 | Apple | 100% 自研 |
| 代工 | TSMC | 唯一量产方（传闻） |
| 竞品 | Qualcomm AR1 | Meta 主力 |
| 封测 | ASE / 苹果自有 | Watch SiP 同类 |

---

## 4. 供应链逻辑

```
IP/EDA → TSMC 晶圆 → 苹果封测/SiP → EMS → 零售 + iPhone
```

---

## 5. 厂商优势

- **Apple**：Watch 十年低功耗经验；无授权费；生态绑定
- **TSMC**：N3E 量产成熟；长期产能合约
- **Qualcomm（对比）**：参考设计完整，但 <1W 平台弱于垂直整合
