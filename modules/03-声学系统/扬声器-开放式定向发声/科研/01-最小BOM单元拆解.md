# 扬声器 · 最小 BOM 单元拆解

> 非官方推演 | 镜腿双侧各 1 路发声通道

## 1. 拆解树

```
镜腿开放式扬声器（×2）
├── 发声单元（动圈 Φ6–8mm 或 平衡电枢 BA）
├── 镜腿音腔塑胶件（前腔/导音管/出音孔座）
└── 阻尼网 + 防尘网（出音孔）
```

## 2. 单元清单

| 最小单元 | 规格/用量 | 供应商候选 | 可再拆? |
|----------|-----------|------------|---------|
| 动圈微型单元 | Φ6–8mm，4–16Ω，镜腿厚度 ≤4mm | 歌尔、瑞声、国光 | 磁路/振膜可拆，采购以模组计 |
| 平衡电枢 BA（备选） | 单动铁，高解析、小体积 | Knowles 系、瑞声 | 否（镜腿高配方案） |
| 音腔塑胶件 | PCTG/PC 注塑，含导音管 | 歌尔一体交付或结构厂 | 否 |
| 阻尼网 | 聚酯/金属网，出音孔 | 标准件 | 否 |

## 3. N50 传闻用量（单副）

| 单元 | 用量 |
|------|------|
