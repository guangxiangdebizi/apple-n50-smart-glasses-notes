# 扬声器 · 镜腿内开放式定向发声

> **免责声明**：产业链推演，**非苹果官方规格**。BOM 见 [`科研/`](./科研/)。

---

## 1. 概述

N50 传闻采用**镜腿内嵌 OWS 类定向扬声器**，与 Meta Ray-Ban、华为 Eyewear 同类。声学 + 结构交叉件：与醋酸纤维镜腿内腔、电池、FPC 共设计。

| 参数 | 传闻/推演 | 置信度 |
|------|-----------|--------|
| 类型 | 开放式定向发声，非骨传导 | 中 |
| 位置 | 双镜腿内，近耳廓 | 中 |
| 供应商 | 歌尔、瑞声 AAC | 中 |

---

## 2. 供应链

```
振膜/磁路 → 微型单元（歌尔/瑞声）→ 镜腿声腔 ID（苹果）→ 组装 → DSP 校准
```

**竞争**：歌尔 LBS 漏音抑制 vs 瑞声高 SNR 麦+扬声器协同；Meta 声学模块 teardown **~$5.5**（含麦+扬声器）。

---

## 3. 未解问题

漏音标准、低频不足、THD 与 **<2W** 功耗冲突。
