## 模块概述

N50 整机组装（EMS）需在**超小型可穿戴**产线上完成 SMT、镜腿穿线、声学调校、镜框–电子件合装与最终 QA。立讯、歌尔、富士康均为苹果 wearable 核心 EMS，传闻 **立讯主装 + 歌尔声学** 分工。

## 供应链逻辑

```
元器件 → SMT → 镜腿预装（电池+FPC穿铰链）→ 镜框合装 → 声学调校 → OQC
```

**与 VP 对比**：VP 单机 EMS ~$150–200；N50 无显示、体积小，但 **FPC 穿铰链良率** 是新风险点。
