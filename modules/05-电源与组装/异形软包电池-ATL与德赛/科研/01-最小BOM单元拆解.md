# 异形软包电池 · 最小 BOM 单元拆解

> **非官方科研估算**：按可穿戴 Pack 供货结构拆分，**非 ATL/德赛报价单**。

## 拆解树

```
L2-异形软包-Pack（单镜腿）
├── 1. 电芯 core（ATL）
│   ├── 正极/负极/隔膜/电解液（计入 $/Wh）
│   ├── 异形叠片 + 冲壳
│   └── 标称 155mAh @ 3.7V ≈ 0.574Wh
├── 2. PCM / BMS IC
│   ├── 过充/过放/过流 MOSFET
│   └── Fuel Gauge（苹果定制概率高）
├── 3. NTC
│   └── 镜腿弯折区温度采样
└── 4. 软包铝塑膜 & 封装
    ├── 铝塑复合膜（lamination）
    ├── 极耳 / 镍片
    └── 保护胶、绝缘 PET
```

## 与 Watch / AirPods 差异

| 维度 | Apple Watch | N50 镜腿 Pack |
|------|-------------|---------------|
| 形态 | 矩形/弧形表壳内 | **S 弯异形**，沿臂长 |
| 容量 | 300–400mAh 级 | **~155mAh/侧** |
| 弯折 | 低 | **铰链区反复形变** → BMS + 结构共设计 |
| 供应商 | ATL + 德赛（VP 亦用德赛） | 传闻延续 |

## 锚点对照

- **$/Wh**：可穿戴软包量产 **$0.08–0.15/Wh**；超薄片 **+30–40%**（Global Growth Insights）。
- **155mAh 参考**：Samsung Jinju（Android XR 泄露）**155mAh**、~50g 整机（TechTimes）。
- **BofA**：AI 眼镜 BOM 中电池 **3–5%**（~$120 整机 → **$3.6–6** 电池带宽）。

## 配置假设

1. **基准**：双腿各 **1×155mAh Pack**（合计 ~310mAh）。
2. **备选**：单腿 **1×280mAh**，本表按双 Pack 计，单 Pack 用量 ×2 汇总见 03。
