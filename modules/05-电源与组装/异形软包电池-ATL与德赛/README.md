## 最小单元链

```
ATL 异形叠片电芯（mAh × $/Wh）
  → PCM/BMS IC（保护 +  fuel gauge）
  → NTC 热敏电阻
  → 软包铝塑膜 / 极耳 / 封装胶
  → 德赛 Pack 集成 + 镜腿结构配合
```

## 传闻规格

| 参数 | 值 | 置信度 |
