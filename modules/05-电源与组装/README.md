# 05 · 电源与组装

> **免责声明**：基于产业链推演与公开报道整理，**非苹果官方 BOM 或供应商确认**。

---

## 模块概述

电源与组装是 N50 能否达成 **<50g 重量**与**全天佩戴**的关键约束模块。镜腿内空间有限，需采用**异形软包锂电**；整机组装则依赖果链 EMS 在微型化、良率、声学调校上的综合能力。

---

## N50 传闻规格

| 子模块 | 传闻要点 | 置信度 |
|--------|---------|--------|
| 电池形态 | 左右镜腿各一或单侧主电，**异形软包** | 中 |
| 电芯供应商 | **ATL**（宁德新能源）、**德赛电池** | 中 |
| 容量 | 未公开；同类参考 ~150–200mAh/侧 | 低 |
| 续航目标 | 传闻对标「数小时」连续 AI 使用 | 低 |
| EMS | **立讯精密**、**歌尔股份**；亦有富士康说法 | 中 |
| 充电 | 充电盒或 USB-C 镜腿触点（待定） | 低 |

## 供应链或系统逻辑

```
ATL（电芯）→ 德赛（Pack/BMS）→ 立讯/歌尔（SMT+整机组装）→ 苹果 QA → 零售
     ↑                                    ↑
  异形叠片工艺                      镜腿穿线+FPC+声学调校
```

**关键约束链**：
1. 镜腿厚度限制电芯截面 → 必须异形叠片，能量密度 > 体积
2. N401 + 双摄像头 + OWS 喇叭共用镜腿空间 → 电池体积为「最后分配」
3. EMS 需同时处理醋酸纤维镜框（结构件）与电子件，装配公差 <0.1mm 级

---

## 优势分析

- **ATL + 德赛**：苹果 wearable 电池「标准组合」，Vision Pro 已验证 Pack 能力
- **立讯**：Vision Pro 独家 EMS 经验可直接迁移至眼镜微型组装
- **歌尔**：若拿下 EMS 或声学，可一体化交付「声–电–结构」
- **三层算力**：主 AI 在 iPhone，眼镜端功耗低，有利于缩小电池
