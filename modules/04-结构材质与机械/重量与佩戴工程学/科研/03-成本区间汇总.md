# 重量与佩戴工程学 · 汇总（重量–成本视角）

## 1. 结论

| 指标 | 说明 |
|------|------|
| **美元 BOM 中位** | **N/A**（非采购模块） |
| **整机重量目标** | **<50 g** |
| **结构+BOM 联动** | 为达标，结构件 BOM 不宜超 **~$14** 中位（见 04 子系统汇总） |

## 2. 关键 tradeoff

```
        成本 ↑
          │    · Ti MIM 铰链
          │  · Mazzucchelli 薄板
          │    · 镁嵌件
    ──────┼────────────────→ 重量 ↓
          │  · 钢铰链（重+便宜）
          │  · 厚 CA（重+省加工）
```

## 3. 与 BOM 模块关系

| 模块 | 重量角色 | 是否计入 04 BOM |
|------|----------|-----------------|

| 铰链 | 1–2 g | 是 |
