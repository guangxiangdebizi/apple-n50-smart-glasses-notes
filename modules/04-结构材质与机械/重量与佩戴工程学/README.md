# 重量与佩戴工程学 · <50 g 约束

> **免责声明**：媒体报道推演。**非苹果官方**。本模块 **科研侧重重量–成本 tradeoff**，非纯 BOM 计价。

---

## 1. 概述

传闻 N50 **<50 g**（中高置信），对标 Meta Ray-Ban Gen2 **48–51 g**。佩戴工程学统筹材质、电子配重、鼻托/镜腿曲线。

| 目标 | 传闻 |
|------|------|
| 整机 | <50 g |
| 对比普通 Ray-Ban | ~45 g |

---

## 2. 重量预算（推演）

| 模块 | 重量 (g) | 成本杠杆 |
|------|----------|----------|
| CA 镜框+镜片 | 25–30 | 减薄 CA ↔ +$/副 良率 |
| 镜腿电子 | 12–18 | 电池 mAh ↔ 克重 |
| 双摄+SoC+声学 | 7–12 | 砍掉第三摄省 ~2g |
| 钛铰链 | 1–2 | Ti→钢 省 ~0.5g、省 ~$0.6 |

**合计 45–52 g** → 解释双摄像传闻。
