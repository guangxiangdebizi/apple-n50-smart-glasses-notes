## 1. 概述

## 2. 供应链

```
木浆/棉 → CA 粒料 → Mazzucchelli 压板 → 定制色板 → CNC（见姊妹模块）
```

**约束**：6–8 mm 块料利用率低；吸湿收缩；镜腿开腔减强度。
