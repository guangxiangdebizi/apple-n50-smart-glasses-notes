# 重量与佩戴工程学 · <50 g 约束

> **免责声明**：重量目标来自媒体报道与产业链推测，**非苹果官方规格**。

---

## 1. 模块概述

智能眼镜的商业化成败，**佩戴重量**常与算力同等关键。传闻 N50 整机目标 **<50 g**（置信度：中高）——与 Meta Ray-Ban Gen 2（**48–51 g**）同量级，确保用户可**全天佩戴**而不产生「头箍感」。

佩戴工程学需统筹：镜框材质、电子件配重、鼻托/镜腿人体曲线、以及**重心位置**（不宜过度靠耳后坠）。

---

## 2. N50 传闻规格

| 参数 | 传闻/推演 | 置信度 |
|------|-----------|--------|
| 整机重量 | **<50 g** | 中高 |
| 对比 | 普通 Ray-Ban Wayfarer ~**45 g** | 高 |
| 电子增重 | 双摄像头 + 电池 + 扬声器 + SoC | 中 |
| 配重策略 | 镜腿对称分布；可能镁合金嵌件 | 低 |
| 发布时间 | 2027 年（Gurman） | 中高 |

## 3. 竞品重量对比

| 产品 | 重量 | 电子配置 | 来源 |
|------|------|----------|------|
| 普通 Ray-Ban Wayfarer | ~45 g | 无 | YouTube 评测 |

| Ray-Ban Meta Display | ~54–69 g | 含 AR 显示 | YouTube / Axis |

**结论**：N50 若达成 <50 g，即**仅比非智能 Ray-Ban 重约 5 g**，属于极高工程挑战。

## 4. 重量预算拆解（推演）

| 模块 | 估算重量 | 备注 |
|------|----------|------|
| CA 镜框 + 镜片 | 25–30 g | 含 Mazzucchelli 板材 |
| 双镜腿（含电池×2?） | 12–18 g | 异形软包锂电（ATL） |
| 摄像头模组×2 | 3–5 g | 索尼 CMOS + 高伟封装 |
| 扬声器 + MEMS×4 | 2–3 g | 歌尔/瑞声微型件 |
| N401 SoC + FPC | 2–4 g | Watch 系低功耗 |
| 铰链 + 金属件 | 1–2 g | 钛 MIM |
| **合计** | **45–52 g** | 需严格减重才能 <50 g |

→ 任何**+2 g**的模块（如第三颗摄像头）都可能打破目标，解释 N50 传闻**仅双摄像**的配置。

## 5. 供应链与工程逻辑

```
苹果 ID / 人机模型（头宽、鼻高分布）
      ↓
材料选型：CA 镜框 + 钛/镁嵌件（减重）
      ↓
镜腿内腔：电池最大体积 vs 扬声器声腔争用
      ↓
配重仿真：重心靠近眉心，减少耳部压迫
      ↓
试产：1000+ 小时佩戴测试 / 运动场景
```

**关键供应商角色**：
- **ATL/德赛**：异形电芯能量密度决定镜腿厚度
- **长盈/领益**：结构件壁厚最小化
- **Mazzucchelli**：轻量 CA 配方 vs 强度

---

## 6. 厂商优势（减重相关）

| 环节 | 优势方 | 原因 |
|------|--------|------|
| 电池 | ATL | 苹果主力；异形软包经验 |
| 结构 | 长盈精密 | 镁合金/3D 打印结构件（Kuo） |
| 板材 | Mazzucchelli | 高密度 CA，可减薄镜腿 |
| 模组 | 高伟电子 | 微型摄像头封装 |

---

## 7. 前沿未解问题

- **处方镜片增重**：高折射率镜片可 +5–10 g，是否标配非处方
- **性别差异**：4 款造型是否覆盖窄脸/宽脸，重量是否分级
- **发热与舒适**：SoC 发热导致镜腿微胀，长期佩戴感知
- **与 AirPods 叠戴**：耳部压力叠加时的 UX 设计
