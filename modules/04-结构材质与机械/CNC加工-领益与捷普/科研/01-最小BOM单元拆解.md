# CNC 加工 · 最小 BOM 单元拆解

## 1. 拆解树

```
镜框 CNC 工序包（单副）
├── 五轴铣削（型腔/铰链位/镜腿槽）
├── 滚筒滚光
└── 加工分钟 × 工厂费率（核心计价单元）
```

## 2. 单元清单

| 最小单元 | 规格 | 供应商 | 可再拆? |
|----------|------|--------|---------|
| 五轴铣削 | 前框+镜腿；15–25 min/副 | 领益、捷普、丹阳 OEM | 按分钟计费 |
| 刀具/夹具摊销 | — | EMS | 已含费率 |
| 滚光 | 8–15 min/副 | 同厂 | 否 |

## 3. N50 用量

| 单元 | 单副 |
|------|------|
| 铣削+滚光 | 1 副；合计 **20–35 加工分钟**（基准 25 min） |

| 锚点 | 参考 |
|------|------|
