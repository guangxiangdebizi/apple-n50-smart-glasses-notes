## 1. 概述

机械+电气交叉：开合 **>1 万次**，**FPC 穿铰链** 连镜框摄像头与镜腿电池。传闻 **钛合金 MIM**，兆威/长盈/中伟等。

| 参数 | 传闻 | 置信度 |
|------|------|--------|
| 材质 | Ti / Ti-6Al-4V | 低–中 |
| 工艺 | MIM + 精加工 | 中 |

---

## 2. 成本逻辑

Ti MIM 加工 **$2–8/件**（行业）；眼镜铰链 **一对** 含左右件+轴芯。量产摊薄模具后 **$0.5–3/对** 与任务给定区间一致。
