# 模组封装 · 高伟电子（Cowell e Holdings）

> **非官方说明**：N50 选用高伟为**产业链推演**；高伟公开披露以 iPhone/iPad **CCM** 为主，智能眼镜模组无官宣。

---

## 1. 模块概述

相机模组（CCM）将 **CIS + 镜头 + 滤光片 + VCM（若有）+ FPC** 封装为可 SMT 的单元。N50 传闻形态为 **竖向椭圆模组 + 隐私指示灯一体化**，对封装厂提出 **异形 COB、良率、散热与 LED 共板** 要求。

**高伟电子（1415.HK）** 隶属立讯体系（立景创新控股 **>70%**），是苹果 **iPhone/iPad 前摄 CCM** 核心供应商，并正切入后摄。

## 4. 供应链逻辑

```
索尼 CIS + 大立光/玉晶光镜头 → 高伟东莞/越南工厂 COB/FC → EMS（立讯/歌尔）
```

**可穿戴延伸逻辑**：

1. 前摄 CCM 经验 → **小尺寸、自拍取向** 与竖向 9:16 主摄相似。
2. 立讯集团 **整机组装 + 模组** 垂直整合，降低 N50 新品类沟通成本。
3. 公司已布局 **VR/AR、激光雷达** 模组（速腾聚创合作），工程路径可复用至眼镜。

**产能**：东莞新自动化工厂 rumored **2026 年 12 月** 完工（研报），匹配 N50 **2027 量产** 时间线（郭明錤，中高置信）。

---

## 5. 厂商优势

1. **果链绑定**：深度参与 iPhone 光学升级（超广角、潜望），工艺与苹果 PQ 标准同步。
2. **FC 产线稀缺性**：倒装芯片封装利于超薄模组（研报多次强调）。
3. **立讯生态**：与立讯精密 EMS、立景创新镜头资源协同。
4. **多摄封装经验**：若 N50 双摄 + LED，类似 iPhone 多摄模组 SMT 流程。
5. **成长弹性**：后摄放量已验证；眼镜 CCM 为 **增量 SKU** 而非替代手机产能。

---

## 6. 前沿未解问题

1. **椭圆模组应力**：COB 胶水圈在非对称外壳下的热膨胀匹配。
2. **隐私 LED 法规**：与相机共板时故障安全（LED 失效是否禁拍）。
3. **无 OIS 良率**：行走模糊靠 EIS，模组厂是否需主动校准 IMU-相机外参。
4. **防水**：醋酸纤维镜框 + 模组接缝 IP 等级。
5. **与 LG Innotek 分工**：苹果是否仅让高伟做眼镜而手机后摄继续双供。
