# 镜头 · 最小 BOM 单元拆解

## 主摄镜组

```
镜筒（椭圆） + 遮光罩
├── Lens G1–G4（塑胶非球面，ASPH）
├── （可选）G5 保护片 / cover
├── spacer 垫圈
└── 固定焦（无 VCM 时无音圈电机）
```

## 副摄镜组

```
简镜筒
├── Lens G1–G2（广角，畸变大）
└── 固定焦
```

## 单颗镜片成本构成

| 子项 | 说明 |
|------|------|
| 塑胶粒子 + 注塑 | PEEK/PMMA 等 |
| 镀膜 | AR / IR-cut 膜层（部分在滤光片） |
| 检测 | MTF、偏心 |
| **模具摊销** | 椭圆镜筒 **$50万–200万** 级模具 ÷ 产量 |

## 3mm 模组溢价逻辑

| 因素 | 对镜头 BOM 影响 |
|------|-----------------|
| 更短 TTL | 镜片曲率更陡，面型难 |
| 椭圆开孔 | 非旋转对称，**每片+$0.05–0.15** |
| 无 OIS | 省 VCM **$1–3**，但 EIS 算力外移 |

## 边界

- **不含** sensor、CCM 贴装、FPC。
- **VCM**：N50 传闻固定焦；若隐藏式 AF，VCM 划入本 L2 或封装 L2（**+$1.2–2.5**）。
