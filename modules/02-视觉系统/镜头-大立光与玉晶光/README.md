## 最小单元链

```
单颗塑胶非球面镜片（注塑+镀膜）
  × N（主摄 4–6 / 副摄 2–3）
  → 镜筒 + 遮光罩（椭圆）
  → 模具摊销（$/副）
```

## 锚点

- 手机镜头组占模组 **15–25%**（IndexBox）。
- Ray-Ban Meta 主摄 DFOV **99°**（Brochure）；**~3mm** 模组厚度竞赛（Camemake / IMX681 分析）。
- 智能眼镜 **3mm 级** 相对 4.5mm 传统：光学更紧 → **镜片良率↓、模具更贵**。
