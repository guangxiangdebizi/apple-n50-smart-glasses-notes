# 副摄 · 最小 BOM 单元拆解

## 拆解树

```
L2-副摄-CIS
├── 1. Sensor die
│   ├── 1–2MP（或 VGA）Rolling shutter CMOS
│   └── 小 die（< Meta OV5675 5MP 面积）
├── 2. 滤光 / 盖板
│   └── 标准 IR cut（规格低于主摄）
├── 3. CSP 载体
│   └── wire bond 或小型 FC（成本低于主摄堆叠）
└── 4. 简镜组（物理上属镜头 L2，此处仅标注接口）
    └── 2–3 片塑胶广角（FOV > 主摄）
```

## 与主摄分工

| 项目 | 主摄 | 副摄 |
|------|------|------|
| 像素 | 5–8MP | 0.3–2MP |
| 堆叠 | 高概率双层/三层 | 单层或简堆叠 |
| 快门 | 卷帘 | 卷帘（手势够用） |
| 常开 | 事件驱动 | **可能 always-on** |

## 锚点

- **OV5675**（5MP）：Meta 副摄×2；若苹果仅 **1–2MP**，die 成本应 **显著低于** $9 级主摄芯片。
- **ams NanEyeC**（320×320）：模组厚 ~2.29mm，说明「感测级」可走超小 CCM（$3–8 整机模组量级参考）。

## 不含

- 双摄标定软件、N401 ISP 算力（计在 SoC 模块）
- MIPI 走线长度（FPC L2）
