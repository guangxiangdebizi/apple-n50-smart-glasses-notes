# 模组封装 · 最小 BOM 单元拆解

## 主摄 CCM 工序物料

```
组装 BOM（value-add）
├── 主动对准（AA）工时摊销
├── 结构件：椭圆支架、铜箔散热垫（可选）
├── 胶粘：UV 胶、环氧、遮光黑胶
├── 柔性尾板（短 FPC，非穿铰链大 FPC）
├── 连接器 / ZIF
├── 隐私 LED（主摄侧）
└── 校准数据烧录（OTP/EEPROM 摊）
```

## 副摄 CCM

- 流程同主摄但 **无 LED 或共用指示灯通道**（传闻未定）。
- AA 精度要求低于主摄 → 工时 **−30–40%**。

## 成本结构（行业）

| 块 | 占成品 CCM 比例 |
|----|-----------------|
| 来料 CIS+镜头 | 70–85%（已在其他 L2） |
| **本 L2 增值** | **15–30%** |

## 3mm 超薄溢价

- FC 倒装、更严平整度 → 良率损失 **+2–5%** 摊销。
- 椭圆支架注塑 + 镜框机械配合 → **结构件 +$0.2–0.5**。

## 不含

- 穿铰链长 FPC（见 FPC L2）
- EMS 整机组装
