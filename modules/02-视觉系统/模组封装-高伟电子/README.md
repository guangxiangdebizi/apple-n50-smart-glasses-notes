## 最小单元链

```
incoming CIS + 镜头
  → AA/active alignment（主动对准）
  → 胶水/围坝 → 镜筒锁附
  → FPC 尾线焊接 / 连接器
  → 隐私 LED 共板
  → 终检（MTF、污点、电流）
```
