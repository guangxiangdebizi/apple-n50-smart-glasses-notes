# FPC · 最小 BOM 单元拆解

## 材料栈

```
FPC stack（6–8L 典型）
├── PI 基材（25µm / 12.5µm）
├── 电解/RA 铜箔（12–18µm 线路）
├── 覆盖膜（Coverlay）
├── 补强：不锈钢 / FR4 局部
├── 表面处理：ENIG / OSP
└── 连接器（BTB/ZIF）×n
```

## 面积估算（单副）

| 区段 | 面积（cm²） | 备注 |
|------|-------------|------|
| 镜框→铰链 | 8–12 | 双 MIPI + 电源 |
| 铰链环状缓冲 | 4–6 | 专利式折叠腔 |
| 镜腿至 SoC | 10–16 | 可与天线共板 |
| **合计** | **22–34** | 可按 **1–2 条** 出货 |

## 按 cm² 成本模型

| 档位 | $/cm²（6L MIPI） | 说明 |
|------|------------------|------|
| 低 | 0.08 | 百万级、标准阻抗 |
| 中 | 0.12 | 动态弯折 RA 铜 + 补强 |
| 高 | 0.18 | 低量 + 严苛可靠性 |

## 可靠性成本

- 动态弯折测试、阻抗 TDR → **+$0.3–0.8/副** 摊入中高位。

## 不含

- 模组短尾线（已在封装 L2 **$0.5** 中位）
