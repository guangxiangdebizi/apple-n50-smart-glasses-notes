## 模块职责

镜框相机 **MIPI CSI-2**、电源、隐私 LED GPIO 经 **钛合金铰链** 至镜腿 N401；承受 **>10k–200k** 次动态弯折。

## 最小单元链

```
PI 基材（25µm 级）+ 铜箔（RA 铜）
  → 覆盖膜 / 补强钢片
  → 阻抗控制 6–8 层
  → 连接器 + 接地
```
