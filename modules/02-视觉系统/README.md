# 02 · 视觉系统 — 子系统总览

> **非官方说明**：本节内容基于 Bloomberg（Mark Gurman）、郭明錤、产业链推演及公开市场数据整理，**不代表苹果 N50 最终规格或供应商名单**。

---

## 1. 模块概述

视觉系统是 N50「无屏 AI 眼镜」的**第一感官入口**：在镜框/镜腿极窄空间内集成**双摄像头**（主摄 + 副摄），完成拍照录像、Visual Intelligence 识物、手势/环境上下文采集，并通过蓝牙/UWB 将图像或特征传至 iPhone 做 Apple Intelligence 主推理。

与 Vision Pro 不同，N50 **无 waveguide 显示**，光学 BOM 几乎全部压在**微型成像链**（传感器 → 镜头 → CCM 封装 → 穿铰链 FPC）上。

```
┌──────────────────────────────────────────────────────────────┐
│  镜框侧：主摄（竖向 9:16）+ 隐私 LED + 椭圆光学开孔          │
│  副摄：低分辨率广角 → CV / 手势 / 环境上下文（端侧预筛选）    │
├──────────────────────────────────────────────────────────────┤
│  镜头层：超微型塑胶镜头（椭圆）— 大立光 / 玉晶光              │
│  传感器：索尼定制微型 CMOS（主 5–8MP / 副更低像素）           │
│  封装：高伟电子 Cowell — 竖向椭圆 CCM + 指示灯一体化           │
│  互连：穿铰链 FPC — 鹏鼎 / 欣兴等果链（置信度低）             │
└──────────────────────────────────────────────────────────────┘
          ↓ MIPI / CSI-2
     N401 SoC（ISP、端侧预筛选）→ iPhone（Neural Engine）
```

## 4. 供应链逻辑（一级）

```
索尼（CMOS）→ 大立光/玉晶光（镜头）→ 高伟（COB/FC 模组）→ 立讯/歌尔 EMS
                    ↑
              鹏鼎/欣兴（FPC 穿铰链至镜腿 SoC/电池）
```

**关键约束**：整机 **<50g**（中高置信）迫使视觉链在 **厚度 ~3–4.5mm 模组** 量级竞争（行业公开对标见子文档），而非手机级 7–10mm CCM。

## 6. 前沿未解问题（子系统级）

1. **竖向 9:16 与椭圆开孔** 的光学像场匹配（CRA、渐晕、边缘解析力）。
2. **双摄 + 隐私 LED** 在醋酸纤维镜框上的热管理与误触发。
3. **穿铰链 FPC** 在数万次弯折后的 MIPI 信号完整性。
4. **端侧 ISP 预筛选 vs 全帧上传** 的隐私、带宽与续航三角。
5. **无 OIS 微型模组** 在行走拍摄下的 EIS 算力归属（N401 vs iPhone）。
