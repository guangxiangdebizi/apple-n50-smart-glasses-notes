# 主摄 CMOS · 索尼（L2）

> **非官方**：N50 主摄 5–8MP、9:16 竖向画幅为产业链推演，非苹果/索尼确认。

## 模块职责

主摄 CIS 是画质与合规核心：拍照、录像、Visual Intelligence 主画面。传闻 **索尼定制 5–8MP**（低于 Ray-Ban Meta **12MP IMX681**），体现「眼镜轻采集、iPhone 重成像」。

## 最小单元链（本 L2 边界）

```
sensor die（堆叠 BI-CMOS）
  → micro-lens / 片上滤光
  → IR/可见光滤光片（盖板级）
  → CSP/COB 基板 + 凸块/引线键合
```

**不含**：塑胶镜筒、VCM、CCM 组装（见「镜头」「模组封装」L2）。
