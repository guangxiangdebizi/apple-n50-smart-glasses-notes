# 苹果 N50 智能眼镜：硬件与供应链详解

> **免责声明**：本文档基于 Bloomberg（Mark Gurman）、郭明錤、MacRumors、OFweek/AsiaICT 等公开信息与产业链推演整理，**不代表苹果官方规格**。具体型号与最终供应商以正式发布为准。

---

## 1. 产品定位与系统架构

### 1.1 是什么 / 不是什么

| 是 | 不是 |
|----|------|
| 无屏 AI 智能眼镜，iPhone 配件 | AR 头显（无 waveguide 光波导） |
| 摄像头 + 麦克风 + 扬声器 + 轻量 SoC | 独立运行的「小电脑」 |
| Apple Intelligence 的前端传感器 | Vision Pro 的廉价版 |

### 1.2 三层算力架构

```
┌─────────────────────────────────────────────────────────────┐
│  眼镜层：采集（看/听）+ 轻算力（唤醒、ISP、音频、预筛选）     │
│         ↕ 蓝牙 + 传闻 UWB 低延迟链路                        │
│  iPhone层：主 AI 推理（Neural Engine / Apple Intelligence） │
│         ↕ 蜂窝 / Wi-Fi                                      │
│  云端层：复杂大模型任务（Siri、部分翻译/问答，按需上云）      │
└─────────────────────────────────────────────────────────────┘
```

**核心逻辑**：眼镜 ≠ 蓝牙管道；眼镜有芯片，但**重 AI 在 iPhone**，云只在需要时参与。

## 2. 眼镜端硬件 BOM（按子系统）

### 2.1 算力与存储

| 模块 | 规格/代号（传闻） | 可能供应商 | 置信度 |
|------|-------------------|------------|--------|
| 主控 SoC | **N401**；Apple Watch **S 系列**魔改；功耗 **<2W** | 苹果设计 | 高 |
| 制程 | 台积电 **3nm / N4P（N3E）** | **TSMC** | 中 |
| 内存 | 定制低功耗 **LPDDR**，SoC 旁共封装 | **三星 / SK海力士** | 中 |
| ISP | 图像信号处理、端侧预筛选 | 苹果自研 | 中 |
| 无线 | **蓝牙** + 传闻 **UWB** | 集成于 N401 / U1 体系 | 中 |

> N401 为内部代号，无公开零售型号。

### 2.2 视觉系统（双摄像头）

| 模块 | 规格（传闻） | 可能供应商 | 置信度 |
|------|--------------|------------|--------|
| 主摄 CMOS | 5–8MP；竖向 **9:16**；HDR；支持 4K 录制说法 | **索尼 Sony** 定制微型 CMOS | 中 |
| 副摄 | 低分辨率广角；CV、手势、环境上下文 | 索尼等 | 中 |
| 镜头 | 超微型塑胶镜头；椭圆开孔 | **大立光**、**玉晶光** | 中高 |
| 模组封装 | 竖向椭圆 + 隐私指示灯 | **高伟电子 Cowell** | 中 |
| FPC 排线 | 穿铰链柔性电路 | 鹏鼎、欣兴等果链 | 低 |

### 2.3 声学

| 模块 | 规格（传闻） | 可能供应商 | 置信度 |
|------|--------------|------------|--------|
| 扬声器 | 镜腿内开放式定向发声 | **歌尔**、**瑞声 AAC** | 中 |
| 麦克风 | **3–4 颗 MEMS**；波束成形 | 歌尔、瑞声 | 中 |

### 2.4 结构 / 材质 / 机械

| 模块 | 规格（传闻） | 可能供应商 | 置信度 |
|------|--------------|------------|--------|

| 铰链 | 钛合金微型铰链 | **兆威机电**、**长盈精密** | 低–中 |
| 重量目标 | **<50g** | — | 中高 |

### 2.5 电源与组装

| 模块 | 规格（传闻） | 可能供应商 | 置信度 |
|------|--------------|------------|--------|
| 电池 | 镜腿内异形软包锂电 | **ATL**、**德赛电池** | 中 |
| EMS 组装 | 整机组装 | **立讯精密**、**歌尔**；亦有 **富士康** 说法 | 中 |

## 3. iPhone 配套端（主大脑）

| 模块 | 作用 | 说明 |
|------|------|------|
| A 系列芯片 + Neural Engine | 视觉 AI、Apple Intelligence | 必须配对 iPhone |
| 蓝牙 / UWB | 与眼镜通信 | 低延迟传视觉数据或特征 |
| 蜂窝 / Wi-Fi | 联网、云端大模型 | 眼镜本身不主打独立上网 |
| iOS 27 | 可穿戴框架、Siri 2.0 | 传闻与眼镜同步 |

## 4. 上下游产业链

### 4.1 产业链示意图

```
上游：台积电 | 索尼 | 大立光/玉晶光 | 三星/SK海力士 | 醋酸纤维板材
          ↓
中游：高伟(模组) | 歌尔/瑞声(声学) | ATL/德赛(电池) | 领益/兆威(结构)
          ↓
苹果：N401 自研 + 结构设计 + Apple Intelligence
          ↓
下游：iPhone 用户 | 开发者 API | 私有云/大模型
```

### 4.2 与 Meta Ray-Ban 芯片路线对比

| 项目 | 苹果 N50 | Meta Ray-Ban Gen 2 |
|------|----------|---------------------|
| 主控 | 自研 N401（Watch 系） | **高通 Snapdragon AR1 Gen 2** |
| 独立性 | 强依赖 iPhone | 相对可独立 |
| 镜框 | 苹果自研醋酸纤维 | **EssilorLuxottica（Ray-Ban）** |
| 显示 | 无 | 无 |

## 5. 传闻功能清单

- 升级版 Siri + Apple Intelligence

- 健康追踪（探索中）
