# 01 · 数据来源与引用规范

## 1. 本库允许写入的「实际数据」类型

| 类型 | 是否可直接写进毛利率表 | 要求 |
|------|------------------------|------|
| 上市公司财报数字 | ✅ | 标注报告期、币种、原文链接 |
| Counterpoint/IDC 经权威媒体引述 | ✅ | 双链：媒体 + 机构名 |
| Wellsenn 经券商 PDF 转载 | ✅ | 标注券商报告名 |
| Gurman/Kuo 传闻 | ⚠️ 仅用于 ASP/时间线情景 | 标注「传闻，非官方」 |
| 本库 BOM 科研模型 | ⚠️ 仅用于 Tier D 情景 | 不得与 A 级混在同一行 |

## 2. 毛利率计算公式（统一）

```
硬件毛利率 = (零售价 ASP − 制造成本 BOM) / ASP × 100%
```

- **ASP**：零售标价（含/不含税需注明；Counterpoint $299 为美国基础版）
- **BOM / 生产成本**：Counterpoint 称 “total **production cost** nearly $135”
- **不含**：渠道分润、营销、R&D 摊销、软件成本 — 若讨论「经营毛利率」需另表
