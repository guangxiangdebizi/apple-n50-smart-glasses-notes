## 1. 软件技术栈

| 层级 | 组件 | 说明 |
|------|------|------|
| 操作系统 | **iOS 27** + 可穿戴扩展 | 传闻 2026–2027 与眼镜协同发布 |
| 端侧 AI | **Apple Intelligence** | 部分推理在 iPhone 本地 |
| 语音 | **Siri 2.0** | 传闻接入 Gemini 等作底层 LLM |
| 视觉 AI | **Visual Intelligence** | 识物、OCR、翻译、地标 |
| 私有云 | **Private Cloud Compute** | 非全部数据上公有云 |
| 连接协议 | 蓝牙 LE、传闻 **UWB** | 与 AirPods / AirTag 同类知识 |

### 1.1 典型交互数据流

1. 眼镜摄像头采集 → N401 做 ISP / 压缩 / 预筛选  
2. 经蓝牙（+UWB）送至 iPhone  
3. iPhone Neural Engine 运行 Visual Intelligence  
4. 复杂问题可选云端大模型  
5. 结果经眼镜扬声器或 AirPods 播报
